## 题目描述

称序列 $a$ 是序列 $b$ 的子序列，如果可以删掉 $b$ 中零个或更多元素使得其与 $a$ 相等。

给定两个序列 $t$ 和 $s$ ，你要求出 $s$ 的最长子序列 $s'$ 的长度，使得 $s'$ 和 $t$ 的最长公共子序列长度不超过 $1$ 。

## 输入格式

第一行，两个正整数 $n$ 和 $m$ ，分别表示 $s$ 和 $t$ 的长度。

第二行，$n$ 个正整数 $s_1,s_2,…,s_n$ 。

第三行，$m$ 个正整数 $t_1,t_2,…,t_m$ 。

## 输出格式

一行一个整数，表示最长子序列 $s′$ 的长度。

---

```
5 4
1 2 3 1 2
3 1 2 3
```

```
2
```

---

2在附件

---

$n,m \le 5\times 10^5$
